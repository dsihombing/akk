Deskripsi Mata Kuliah

Mata kuliah ini memperkenalkan pendekatan komputasional mutakhir dalam studi komunikasi. Mahasiswa akan dilatih untuk mengumpulkan, memproses, dan menganalisis data jejak digital (digital traces) dari media sosial maupun portal berita daring. Pembelajaran dirancang untuk mengombinasikan landasan teori komunikasi klasik, pengantar pemrograman bahasa R/Python, hingga implementasi teknik analisis tingkat lanjut seperti Pemodelan Topik (Topic Modeling) dan Analisis Jejaring Sosial (SNA).

Modul Pembelajaran Interaktif

Praktikum Dasar: Persiapan IDE

Pengenalan ekosistem Cloud Computing (Google Colab), lingkungan IDE lokal (RStudio/Jupyter), dan translasi logika pemrograman.

Mulai Orientasi

Suplemen: Kompilasi Skrip

Direktori referensi (cheat sheet) berisi potongan kode Python dan R yang siap disalin untuk eksperimentasi di Google Colab.

Akses Skrip Dasar

Minggu 1: Pengantar & Etika

Memahami The Computational Turn, karakteristik jejak digital, alur riset integratif, serta etika privasi di era Post-API.

Akses Modul

Minggu 2: Dasar Pemrograman

Penguasaan logika sekuensial, taksonomi variabel, konstruksi DataFrame, dan simulasi penulisan skrip R/Python.

Akses Modul

Minggu 3: Manajemen Data

Mendiagnosis bias invisible engagement, arsitektur JSON/CSV, dan implementasi Analisis Data Eksploratif (EDA).

Akses Modul

Minggu 4: Pra-pemrosesan Teks

Implementasi NLP: tokenisasi, lemmatisasi, stop-word removal, dan transisi paradigma menuju Transfer Learning.

Akses Modul

Minggu 5-6: Frekuensi & Sentimen

Ekstraksi wacana via N-Gram, kalibrasi algoritma TF-IDF, serta klasifikasi polaritas emosi publik (Analisis Sentimen).

Akses Modul

Minggu 7: Topic Modeling

Ekstraksi tema laten tak terawasi (Unsupervised) menggunakan LDA, serta disrupsi pemodelan topik visual (gambar/video).

Akses Modul

Minggu 8: UTS

Ujian Tengah Semester: Evaluasi komprehensif atas penguasaan teoretis dan kapabilitas praktikum pemrograman dasar.

Menunggu Jadwal

Minggu 9: Web Scraping & API

Teknik ekstraksi data dari situs berita daring dan penarikan otomatis jejak digital melalui API media sosial resmi.

Modul Terkunci

Minggu 10-11: Analisis Jejaring

Social Network Analysis (SNA): Mengalkulasi sentralitas aktor, mendeteksi komunitas, dan visualisasi graf jaringan.

Modul Terkunci

Minggu 12-13: Machine Learning

Pengantar pembelajaran mesin untuk klasifikasi data, klasterisasi opini publik, dan prediksi algoritma otomatis.

Modul Terkunci

Minggu 14: Publikasi Karya

Translasi temuan komputasional ke dalam manuskrip artikel ilmiah, laporan penelitian, atau presentasi visual interaktif.

Modul Terkunci

Minggu 15: Presentasi Proyek

Diseminasi akhir proyek komputasional mahasiswa, proses telaah sejawat (peer-review), dan refleksi akademis.

Modul Terkunci

Minggu 16: UAS

Ujian Akhir Semester: Validasi pamungkas atas kemampuan mahasiswa dalam mengeksekusi riset komunikasi komputasional.
