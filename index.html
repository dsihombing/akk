Deskripsi Mata Kuliah

Mata kuliah ini memperkenalkan pendekatan komputasional dalam studi komunikasi. Mahasiswa akan mempelajari cara mengumpulkan, mengolah, dan menganalisis data jejak digital dari media sosial, berita daring, dan platform lainnya. Pembelajaran mengombinasikan teori komunikasi klasik, pengantar pemrograman (R/Python), hingga teknik analisis tingkat lanjut seperti Topic Modeling dan Social Network Analysis.

Modul Pembelajaran Interaktif

Minggu 1: Pengantar & Etika

Memahami teori komputasional, langkah-langkah analisis, serta etika privasi dan anonimisasi data.

Mulai Belajar

Minggu 2: Dasar Pemrograman

Pengenalan logika pemrograman dasar, tipe variabel, struktur data, dan pengenalan R/Python.

Mulai Belajar

Minggu 3: Data Komunikasi

Mengidentifikasi jenis data (terstruktur/tidak terstruktur), format data (CSV/JSON), dan membaca data jejak digital.

Segera Hadir

Minggu 4: Pra-pemrosesan Teks

Teknik tokenisasi, lemmatisasi, penghapusan stop-word, dan pembersihan data teks komunikasi.

Segera Hadir

Minggu 5-6: Frekuensi & Sentimen

Analisis frekuensi kata, perhitungan n-gram, TF-IDF, serta penerapan analisis sentimen dasar pada teks.

Segera Hadir

Minggu 7: Topic Modeling

Mengekstraksi dan menemukan tema/topik pembicaraan tersembunyi dari teks skala besar menggunakan metode LDA.

Segera Hadir

Minggu 8: UTS

Ujian Tengah Semester: Evaluasi pemahaman konsep dan praktik pemrograman dasar yang telah dipelajari.

Jadwal Menyusul

Minggu 9: Web Scraping & API

Teknik pengambilan data dari situs berita dan penarikan otomatis jejak digital melalui API media sosial.

Segera Hadir

Minggu 10-11: Analisis Jejaring Sosial

SNA (Social Network Analysis): Memahami graf, aktor kunci (sentralitas), dan visualisasi jaringan wacana.

Segera Hadir

Minggu 12-13: Machine Learning

Pengantar pembelajaran mesin untuk komunikasi: klasifikasi data, klasterisasi, dan prediksi otomatis berbasis algoritma.

Segera Hadir

Minggu 14: Publikasi Karya

Menyusun temuan hasil analisis ke dalam bentuk laporan penelitian, artikel ilmiah, atau dokumen interaktif.

Segera Hadir

Minggu 15: Presentasi Proyek

Presentasi proyek komputasional mahasiswa, diskusi kelas, dan sesi refleksi proses pembelajaran.

Segera Hadir

Minggu 16: UAS

Ujian Akhir Semester: Validasi akhir kemampuan mahasiswa secara komprehensif dalam riset berbasis data.

Jadwal Menyusul
