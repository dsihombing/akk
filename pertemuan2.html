Dasar Pemrograman (R & Python)

Membangun fondasi logika komputasional, memahami variabel, struktur data sederhana, dan menulis skrip analitis pertama Anda.

Berpikir Seperti Komputer (Computational Thinking)

Sebagai peneliti komunikasi, Anda terbiasa membaca nuansa, konteks, dan emosi dalam sebuah teks. Namun, komputer tidak memiliki empati. Ia hanya memahami instruksi yang sangat spesifik dan logis. Dasar dari instruksi tersebut bertumpu pada tiga logika utama (Control Flow):

1. Sequence (Urutan)

Komputer membaca kode dari atas ke bawah, baris demi baris. Jika Anda meminta komputer memvisualisasikan grafik sebelum Anda memerintahkannya mengunduh data, program akan error.

2. Selection (Pemilihan / Kondisi If-Else)

Membuat keputusan berdasarkan kondisi tertentu.
Contoh Komunikasi: JIKA (IF) sebuah tweet mengandung kata "kecewa", MAKA tandai tweet tersebut sebagai sentimen "Negatif". SELAIN ITU (ELSE), tandai sebagai "Positif/Netral".

3. Iteration (Perulangan / Loops)

Inilah kekuatan utama pemrograman! Alih-alih menghitung frekuensi kata pada 10.000 artikel secara manual, Anda bisa menggunakan perintah FOR atau WHILE untuk mengulang instruksi yang sama pada seluruh dokumen dalam hitungan detik.

Variabel dan Tipe Data

Variabel adalah "wadah" atau kotak penyimpanan di dalam memori komputer. Setiap kotak harus memiliki nama, dan isi dari kotak tersebut ditentukan oleh Tipe Data.

Dalam riset komunikasi, kita sering berhadapan dengan tipe data berikut:

Tipe Data Keterangan Contoh di R / Python
String / Character Teks. Selalu diapit tanda kutip (" "). tweet_teks = "Berita hari ini sangat mengejutkan!"
Integer Angka bulat (tanpa desimal). Sering digunakan untuk metrik sosial. jumlah_retweet = 1500
Float / Numeric Angka desimal. engagement_rate = 4.56
Boolean / Logical Hanya memiliki 2 nilai: Benar (TRUE) atau Salah (FALSE). is_viral = TRUE

Struktur Data Sederhana

Dalam penelitian, kita hampir tidak pernah menganalisis hanya satu buah data tunggal (seperti satu tweet saja). Kita menganalisis ribuan data sekaligus. Karena itu, variabel-variabel di atas harus dikumpulkan ke dalam Struktur Data.

1. Vector (di R) / List (di Python)

Kumpulan data satu dimensi (deret). Bayangkan seperti satu kolom di dalam Excel.
Contoh Kumpulan Umur: c(20, 25, 22, 19, 30)

2. DataFrame (Tabel)

Ini adalah struktur data paling penting dalam riset ilmu sosial kuantitatif! DataFrame adalah gabungan dari beberapa Vector yang membentuk tabel 2 dimensi (memiliki baris dan kolom).

  • Baris (Rows) merepresentasikan Observasi/Kasus (Misal: 1 baris = 1 akun pengguna).
  • Kolom (Columns) merepresentasikan Variabel (Misal: kolom Umur, kolom Pekerjaan, kolom Sentimen).

Cobalah Kode R Berikut Ini:

Salin blok kode di bawah ini, tempelkan ke dalam Cell di Google Colab Anda, dan tekan ikon Play (atau Shift + Enter).

# 1. MEMBUAT VARIABEL DAN TIPE DATA
judul_berita <- "Pemilu 2024: Dinamika Komunikasi Politik di Media Sosial"
jumlah_komentar <- 450
is_hoax <- FALSE

# Mencetak isi variabel ke layar
print(judul_berita)

# 2. MEMBUAT DATA FRAME (TABEL RISET)
# Pertama, kita buat Vector (kolom)
nama_akun <- c("UserA", "UserB", "UserC")
sentimen_tweet <- c("Positif", "Negatif", "Netral")
jumlah_likes <- c(120, 15, 45)

# Kedua, kita gabungkan menjadi Tabel (DataFrame)
data_medsos <- data.frame(
  Akun = nama_akun,
  Sentimen = sentimen_tweet,
  Likes = jumlah_likes
)

# Menampilkan tabel
print(data_medsos)

# 3. MENGGUNAKAN FUNGSI SEDERHANA
# Menghitung rata-rata (mean) dari kolom Likes di dalam data_medsos
rata_rata_likes <- mean(data_medsos$Likes)
cat("Rata-rata likes dari seluruh akun adalah:", rata_rata_likes)
