Data dalam Studi Komunikasi Komputasional

Mengenal anatomi jejak digital, format pertukaran data, dan teknik dasar mengeksplorasi dataset menggunakan R dan Python.

Memahami Jejak Digital (Digital Traces)

Dalam metode kuantitatif tradisional, peneliti menggunakan kuesioner yang menghasilkan data angka (skala Likert, dll). Dalam riset komputasional, kita meneliti jejak digital yang ditinggalkan pengguna secara sadar maupun tidak sadar di internet. Jejak ini secara umum terbagi menjadi dua kelompok besar:

1. Data Terstruktur (Structured Data)

Data yang sangat terorganisir, memiliki model yang tetap, dan mudah dimasukkan ke dalam basis data relasional atau tabel Excel. Komputer sangat menyukai data jenis ini karena mudah dihitung.

Contoh dalam Komunikasi: Jumlah followers, jumlah likes, retweets, tanggal pembuatan akun, dan durasi menonton video.

2. Data Tidak Terstruktur (Unstructured Data)

Data yang tidak memiliki format baku, tidak bisa langsung dimasukkan ke dalam kolom angka Excel, dan membutuhkan pemrosesan bahasa alami (Natural Language Processing) atau Computer Vision agar bisa dianalisis.

Contoh dalam Komunikasi: Teks caption Instagram, artikel portal berita, foto meme politik, rekaman podcast, dan video TikTok.

Catatan: Data komunikasi modern hampir selalu bersifat Semi-Terstruktur. Sebuah postingan di platform X (Twitter) memuat teks opini (tidak terstruktur) yang menempel pada data jumlah retweets dan tanggal unggah (terstruktur).

Format Penyimpanan dan Pertukaran Data

Ketika Anda mengambil data dari internet (melalui Web Scraping atau API), data tersebut akan diunduh dalam format-format tertentu. Tiga format yang paling sering dijumpai adalah:

1. CSV (Comma-Separated Values)

Format plain-text yang paling populer untuk menyimpan tabel data. Nilai pada setiap kolom dipisahkan oleh tanda koma (,). Ukuran filenya sangat kecil dan bisa dibuka langsung menggunakan Microsoft Excel, R, maupun Python.

2. JSON (JavaScript Object Notation)

Format yang menyimpan data dalam bentuk key-value pairs (pasangan kunci dan nilai). Format ini sangat fleksibel untuk menyimpan data bersarang (hierarkis) dan merupakan standar emas yang digunakan oleh hampir seluruh API web modern (seperti API YouTube, Reddit, dan platform berita).

{
  "user": "Denny_JC",
  "followers": 1500,
  "tweet": {
    "teks": "Belajar komputasional itu menyenangkan!",
    "likes": 45
  }
}

3. XML (eXtensible Markup Language)

Format lama yang menggunakan struktur tag yang mirip dengan HTML. Saat ini perlahan mulai ditinggalkan karena cukup berat dan memakan banyak karakter (verbose), namun masih sering dijumpai jika Anda menarik data dari RSS Feed portal berita.

Praktik: Membaca Data (Importing)

Sebelum bisa dianalisis, data CSV atau JSON yang telah Anda unduh harus dimasukkan ke dalam memori perangkat lunak analitik Anda (menjadi DataFrame).

A. Menggunakan R (Tidyverse)

Di ekosistem R, paket readr (bagian dari tidyverse) adalah alat terbaik dan tercepat untuk membaca data tekstual.

# Memanggil pustaka utama
library(tidyverse)

# Mengimpor file CSV ke dalam variabel (DataFrame)
# Kita anggap file bernama "data_berita.csv" ada di folder yang sama
df_berita <- read_csv("data_berita.csv")

# Mengimpor file CSV langsung dari URL/Internet
df_covid <- read_csv("https://contoh-dataset.com/tweet_covid.csv")

B. Menggunakan Python (Pandas)

Di Python, perpustakaan utama untuk mengolah data tabular adalah pandas.

# Memanggil pustaka pandas
import pandas as pd

# Mengimpor file CSV
df_berita = pd.read_csv("data_berita.csv")

# Pandas juga sangat handal untuk membaca file JSON
df_komentar = pd.read_json("data_komentar.json")

Exploratory Data Analysis (EDA) Dasar

Setelah data berhasil diimpor, pantang bagi seorang peneliti untuk langsung melakukan analisis sentimen atau visualisasi. Anda harus "berkenalan" dengan anatomi data Anda terlebih dahulu untuk memastikan tidak ada kolom yang rusak atau kosong.

Sintaks Pemeriksaan Cepat di R

# 1. Melihat 6 baris pertama untuk mengintip isi data
head(df_berita)

# 2. Melihat struktur data (jumlah baris, jumlah kolom, dan tipe data per kolom)
glimpse(df_berita)
# atau menggunakan fungsi bawaan dasar: str(df_berita)

# 3. Menghasilkan ringkasan statistik dasar (Nilai Min, Max, Rata-rata)
summary(df_berita)

Sintaks Pemeriksaan Cepat di Python

# 1. Melihat 5 baris pertama
print(df_berita.head())

# 2. Melihat informasi ringkas tipe data dan keberadaan nilai kosong (Missing Values)
print(df_berita.info())

# 3. Menghasilkan ringkasan statistik deskriptif untuk kolom angka
print(df_berita.describe())

Uji Pemahaman
