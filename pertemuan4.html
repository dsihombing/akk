Pra-pemrosesan Teks (Text Pre-processing)

Mempersiapkan data teks mentah ("kotor") menjadi data yang terstruktur ("bersih") agar dapat dipahami oleh algoritma komputer.

Mengapa Teks Perlu Dibersihkan?

Dalam ilmu komputer, terdapat hukum emas yang disebut GIGO (Garbage In, Garbage Out). Jika Anda memasukkan "sampah" ke dalam algoritma, maka hasil analisis yang keluar juga berupa "sampah".

Bagi manusia, kalimat "Korupsi itu JAHAT!!!" dan "korupsi itu jahat" bermakna sama. Namun bagi mesin, huruf kapital, huruf kecil, spasi ekstra, dan tanda seru dianggap sebagai karakter matematis yang sama sekali berbeda.

Tujuan Pra-pemrosesan:

Menyeragamkan struktur teks dan membuang elemen yang tidak memiliki "makna analitis" (seperti URL, angka acak, atau emoji), sehingga mesin hanya berfokus menghitung dan memodelkan kata-kata inti yang relevan dengan pertanyaan riset Anda.

Pipeline (Alur) Pra-pemrosesan Teks

Langkah-langkah ini secara teknis dikenal sebagai teknik dasar dalam Natural Language Processing (NLP).

1

Case Folding & Cleaning

Mengubah seluruh teks menjadi huruf kecil (lowercasing) dan menghapus elemen pengganggu seperti tanda baca (punctuation), angka, simbol, tautan web (URL), dan mentions (@).

Input: "Cek link ini http://berita.com, SANGAT parah! @jokowi"
Output: "cek link ini sangat parah"
2

Tokenisasi (Tokenization)

Memecah sebuah kalimat (string) yang panjang menjadi unit-unit kata yang berdiri sendiri (disebut token). Mesin tidak bisa menganalisis kalimat utuh, ia menganalisis daftar kata.

Input: "cek link ini sangat parah"
Output: ["cek", "link", "ini", "sangat", "parah"]
3

Penghapusan Stop-word (Stop-word Removal)

Menghapus kata-kata hubung atau kata umum yang sangat sering muncul tetapi tidak membawa makna yang signifikan bagi analisis (misal: "dan", "atau", "yang", "di", "ke", "ini").

Input: ["cek", "link", "ini", "sangat", "parah"]
Output: ["cek", "link", "parah"]
4

Lemmatisasi / Stemming

Mengembalikan kata yang memiliki imbuhan ke bentuk kata dasarnya (root word). "Stemming" memotong imbuhan secara kasar, sedangkan "Lemmatisasi" menggunakan kamus bahasa untuk mencari bentuk dasar yang benar secara gramatikal.

Input: ["mempermainkan", "pekerjaan", "terbesar"]
Output: ["main", "kerja", "besar"]

Praktik di R: Menggunakan `quanteda`

Dalam ekosistem R, package quanteda (Quantitative Analysis of Textual Data) adalah salah satu perangkat paling mutakhir, sangat cepat, dan direkomendasikan untuk analisis teks komunikasi.

# 1. Install dan panggil package (jalankan install.packages("quanteda") jika belum ada)
library(quanteda)

# 2. Siapkan data teks mentah
teks_mentah <- c(
  "Vaksinasi COVID-19 sangat PENTING untuk masyarakat!!! 💉 Kunjungi https://vaksin.go.id",
  "Pemerintah harus bertindak TEGAS... @kemenkesRI #sehat"
)

# 3. Jadikan Corpus (kumpulan dokumen berstruktur)
korpus_data <- corpus(teks_mentah)

# 4. Tokenisasi dan Pembersihan (Cleaning & Lowercasing otomatis terjadi di sini)
token_data <- tokens(korpus_data, 
                     remove_punct = TRUE,   # Hapus tanda baca
                     remove_numbers = TRUE, # Hapus angka
                     remove_url = TRUE,     # Hapus tautan web
                     remove_symbols = TRUE) # Hapus simbol/emoji

# 5. Penghapusan Stop-word (menggunakan kamus bahasa Indonesia)
# Catatan: Perlu package 'stopwords' terinstall di R
token_bersih <- tokens_remove(token_data, pattern = stopwords("indonesian"))

# Tampilkan hasil
print(token_bersih)

Praktik di Python: `nltk` & `spaCy`

Di Python, nltk (Natural Language Toolkit) sangat populer untuk akademik, sedangkan spaCy digunakan untuk performa industri yang lebih modern (terutama untuk lemmatisasi tingkat lanjut).

# CONTOH DENGAN NLTK
import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
import re

# Download kamus jika baru pertama kali:
# nltk.download('punkt'); nltk.download('stopwords')

teks_mentah = "Vaksinasi COVID-19 sangat PENTING untuk masyarakat!!! https://vaksin.go.id"

# 1. Cleaning & Lowercasing (Regex)
teks_bersih = re.sub(r'http\S+|[^a-zA-Z\s]', '', teks_mentah)
teks_bersih = teks_bersih.lower()

# 2. Tokenisasi
token_teks = word_tokenize(teks_bersih)

# 3. Hapus Stop-word (Bahasa Indonesia)
stop_words = set(stopwords.words('indonesian'))
token_final = [kata for kata in token_teks if kata not in stop_words]

print("Hasil NLTK:", token_final)
