Python: Logika, Variabel, dan Operasi Dasar

Python dikenal dengan sintaksisnya yang sangat intuitif dan menyerupai bahasa Inggris manusia. Skrip di bawah ini mendemonstrasikan fungsi cetak, penyimpanan variabel, serta komputasi aritmatika.

Python 
# 1. Menampilkan teks ke layar (Fungsi Print)
print("Halo Dunia! Saya sedang mengeksekusi Python di Google Colab.")

# 2. Mendeklarasikan Variabel (Wadah Penyimpanan)
nama_platform = "X (Twitter)"
jumlah_retweet = 450
jumlah_likes = 1200

# 3. Operasi Aritmatika (Menghitung Total Interaksi)
total_interaksi = jumlah_retweet + jumlah_likes

# 4. Menggabungkan teks dan hasil komputasi
print("Platform yang dianalisis:", nama_platform)
print("Total agregasi interaksi publik adalah:", total_interaksi)

Struktur Data: List dan Perulangan (Loop)

List digunakan untuk menampung banyak data dalam satu variabel. Loop digunakan untuk mengeksekusi komputasi secara berulang.

Python 
# 1. Membuat List (Daftar Sentimen)
daftar_sentimen = ["Positif", "Negatif", "Netral", "Positif"]

# 2. Menghitung panjang List (Berapa banyak data yang ada?)
jumlah_data = len(daftar_sentimen)
print("Terdapat", jumlah_data, "data sentimen yang diekstraksi.")

# 3. Perulangan (Mencetak isi List satu per satu)
print("--- Detail Sentimen ---")
for s in daftar_sentimen:
    print("Sentimen terdeteksi:", s)

Python: Memanajemen Data Tabular (Pandas)

Dalam analisis komputasional, struktur data tabular dua dimensi adalah instrumen wajib. Di Python, kita memanggil paket pustaka (library) bernama Pandas untuk mengonstruksi dan membedah DataFrame.

Python (Pandas) 
# 1. Memanggil pustaka manipulasi data (Pandas)
import pandas as pd

# 2. Mendeklarasikan data dalam format 'Dictionary'
data_mentah = {
    "Nama_Politisi": ["Kandidat A", "Kandidat B", "Kandidat C"],
    "Partai": ["Merah", "Biru", "Hijau"],
    "Engagement_Rate": [4.5, 8.2, 3.1]
}

# 3. Mengonversi Dictionary menjadi DataFrame (Tabel)
df_komunikasi = pd.DataFrame(data_mentah)

# 4. Menampilkan bentuk tabel yang rapi
print("--- Struktur Matriks Data ---")
print(df_komunikasi)

# 5. Mengalkulasi rata-rata (mean) dari kolom khusus
rata_engagement = df_komunikasi["Engagement_Rate"].mean()
print("\nRata-rata Engagement Rate adalah:", rata_engagement)

R: Logika, Variabel, dan Operasi Dasar

Bahasa pemrograman R sangat andal untuk fungsi statistika. Berbeda dengan Python yang menggunakan tanda sama dengan (=), R lazim menggunakan tanda panah (<-) untuk memasukkan nilai ke dalam variabel (assignment operator).

R Script 
# 1. Menampilkan teks ke layar (Print & Cat)
print("Halo! R siap mengeksekusi komputasi statistik.")

# 2. Mendeklarasikan Variabel (Assignment dengan panah)
nama_media <- "Portal Berita X"
berita_hoaks <- 15
berita_valid <- 145

# 3. Operasi Aritmatika (Menghitung Total Publikasi)
total_publikasi <- berita_hoaks + berita_valid

# 4. Menggabungkan teks dan hasil komputasi menggunakan 'cat'
cat("Analisis pada", nama_media, "\n")
cat("Total agregasi berita yang dianalisis:", total_publikasi)

Struktur Data: Vektor (Vector)

Dalam R, entitas pengelompokan data dasar disebut Vektor. Vektor dibuat menggunakan fungsi kombinasi c().

R Script 
# 1. Membuat Vektor (Kumpulan Umur Responden)
umur_responden <- c(21, 25, 23, 30, 19)

# 2. Menghitung jumlah elemen (Length)
jumlah_orang <- length(umur_responden)
cat("Total sampel observasi:", jumlah_orang, "\n")

# 3. Menggunakan fungsi statistik bawaan secara instan!
cat("Umur termuda:", min(umur_responden), "\n")
cat("Umur tertua:", max(umur_responden), "\n")
cat("Rata-rata umur:", mean(umur_responden))

R: Memanajemen Data Tabular (DataFrame)

Salah satu keunggulan absolut R adalah kemampuannya membangun kerangka data (DataFrame) secara natif tanpa perlu menginstal paket tambahan. DataFrame adalah gabungan dari beberapa kolom Vektor.

R Script 
# 1. Mendeklarasikan vektor independen untuk kolom tabel
akun_brand <- c("Brand_A", "Brand_B", "Brand_C")
kategori <- c("Fashion", "Teknologi", "Kuliner")
sentimen_positif <- c(850, 120, 340)

# 2. Mengintegrasikan vektor menjadi DataFrame
df_merek <- data.frame(
  Akun = akun_brand,
  Industri = kategori,
  Total_Positif = sentimen_positif
)

# 3. Menampilkan matriks data
print("--- Matriks Evaluasi Brand ---")
print(df_merek)

# 4. Mengeksekusi diagnostik struktur data (Exploratory Data Analysis dasar)
print("\n--- Ringkasan Statistika ---")
summary(df_merek)
